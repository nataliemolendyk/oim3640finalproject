# Description
We decided to create a Flask app for flights departing from Boston Logan Airport. This app uses Python code as well as APIs. The app calculates the drive time from the starting location that the user inputs and converts the location into coordinates to find the distance between that and the airport. A route is then traced with the starting and end points markers and the drive time is calculated based on the distance. The user is asked to input the destination of the flight and the flights going to that location are displayed as well as their status. At the top of the page, the next flight going to the destination is displayed.

# Setup Instructions & Usage
The user should first run the app.py file and a link would be provided in the terminal. The user should click on the link and add in \flight in the url bar and refresh. The flight dashboard will then be displayed and the user should input their starting location and the destination of the flight and press the search bar to display the flight information.
