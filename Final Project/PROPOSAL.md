# What are you building? 
We are building a web application that uses Python code and lets the user input the flight number of their flight. The web application will output the terminal at the Boston Logan Airport and gate as well as the time it takes to drive to that terminal based on an inputed starting location.

# Why? What motivates you to build this? What problem does it solve?
We wanted to build this web application because we feel like going to the airport is stressful enough and having things in multiple places doesn't help. This web application will have all of that necessary information all in one to decrease stress.

# MVP vs. Stretch Goals: What is the minimum working version? What would you add if you have time?
The minimum working version is finding the flight number, the terminal and gate, and finding how long it takes to get there. If we had more time, we would expand it to any airport and add if the flight is running late or early/the time till it boards.
